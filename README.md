# Análisis del Dataset NSL-KDD

## Descripción
Proyecto de análisis y preparación del dataset NSL-KDD para machine learning, incluyendo división estratificada de datos.

## Archivos del Proyecto
- `07_Division_Data_Set.ipynb` - Notebook principal con el análisis
- `07_Division_Data_Set.html` - Versión HTML del notebook
- `index.html` - Página web del proyecto

## Contenido Técnico
- Lectura y exploración del dataset NSL-KDD
- División en conjuntos de entrenamiento, validación y prueba
- Particionado aleatorio y stratified sampling
- Función automatizada para división de datasets
- Visualización de distribuciones

## Tecnologías
- Python 3
- Jupyter Notebook
- Pandas, Scikit-learn
- Matplotlib

## Uso
Abrir `index.html` en el navegador para acceder al análisis completo.
